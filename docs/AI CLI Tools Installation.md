This workshop uses AI-powered command-line interface (CLI) tools to assist in code generation and editing. Please follow the links below to install the necessary tools for your environment.

## Gemini CLI

The Gemini CLI allows you to interact with Google's Gemini models from your terminal and within VS Code.

- **Installation Guide:** [https://github.com/google-gemini/gemini-cli](https://github.com/google-gemini/gemini-cli)
- **VS Code Extension:** Search for "Gemini CLI Companion" in the Visual Studio Code Marketplace. Or go to https://marketplace.visualstudio.com/items?itemName=Google.gemini-cli-vscode-ide-companion and install it directly.

## GitHub Copilot CLI
GitHub Copilot CLI is available with the GitHub Copilot Pro, GitHub Copilot Pro+, GitHub Copilot Business and GitHub Copilot Enterprise plans. GitHub Copilot CLI brings the power of Copilot to your terminal.

- **Installation Guide:** [https://docs.github.com/en/copilot/how-tos/set-up/install-copilot-cli](https://docs.github.com/en/copilot/how-tos/set-up/install-copilot-cli)
- **Usage: ** [https://docs.github.com/en/copilot/how-tos/use-copilot-agents/use-copilot-cli](https://docs.github.com/en/copilot/how-tos/use-copilot-agents/use-copilot-cli)

**Notes:**
Gemini CLI needs to connects with your Google account, while GitHub Copilot CLI needs to connect with your GitHub account. 
- Gemini provides Free tier: 60 requests/min and 1,000 requests/day with personal Google account.
- GitHub Copilot CLI requires GitHub Copilot Pro (300 per month	per month) subscription which is free for all educational students. See documents from https://docs.github.com/en/copilot/how-tos/manage-your-account/get-free-access-to-copilot-pro
