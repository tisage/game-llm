## üöÄ Phase 1: "AI Understands" (5 minutes)

### Prompt 1: Concept & Quick Implementation
```

Requirements:
- Single file implementation (snake_game.py)
- WASD controls
- 600x400 window with 20x15 grid
```

## üîß Phase 2: "AI Fixes Problems" (7-8 minutes)
**Goal**: Show AI debugging capabilities

### Prompt 2: Control Debugging (Use if WASD doesn't work)
```
The WASD controls are not working properly in the Snake game. The snake doesn't move when I press keys. 

Please debug and fix:
- Key event handling
- Make sure WASD keys control the snake movement
- Ensure the game window has proper focus
- Make controls responsive and immediate

Fix the control issues and make sure the snake moves smoothly with WASD.
```

### Prompt 3: Quick Validation (Always use)
```
Please verify the game works correctly:
- Test that WASD moves the snake in correct directions
- Ensure food appears and can be eaten
- Check that score updates properly
- Confirm collision detection works (walls and self-collision)
- Make sure the game is smooth and bug-free

Add any quick fixes needed to make it demo-ready.
```

**Expected outcome**: Fully functional game with working controls  
**Student impact**: "AI actually debugs and fixes problems!"
